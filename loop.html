Looping in JS

Introduction to Loops

In JavaScript, loops are used to execute a block of code repeatedly until a certain condition is met. There are several types of loops available in JavaScript:

  • for loop
  • while loop
  • do...while loop
  • for...in loop
  • for...of loop (Introduced in ES6)

Examples of Loops

1. for Loop


        for (let i = 0; i < 5; i++) {
            console.log(i);
        }

2. while Loop


        let i = 0;
        while (i < 5) {
            console.log(i);
            i++;
        }

3. do...while Loop


        let i = 0;
        do {
            console.log(i);
            i++;
        } while (i < 5);

4. for...in Loop


        const person = {
            name: 'John',
            age: 30,
            gender: 'male'
        };

        for (let key in person) {
            console.log(key + ': ' + person[key]);
        }

5. for...of Loop


        const array = [1, 2, 3, 4, 5];
        for (let value of array) {
            console.log(value);
        }
