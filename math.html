Math and String Manipulation in JavaScript

Math Operations

JavaScript provides built-in functions for performing mathematical operations:

  • Math.abs(): Returns the absolute value of a number.
  • Math.ceil(): Rounds a number up to the nearest integer.
  • Math.floor(): Rounds a number down to the nearest integer.
  • Math.max(): Returns the maximum of two or more numbers.
  • Math.min(): Returns the minimum of two or more numbers.
  • Math.pow(): Returns the base to the exponent power.
  • Math.random(): Returns a random number between 0 and 1.
  • Math.round(): Rounds a number to the nearest integer.
  • Math.sqrt(): Returns the square root of a number.

String Manipulation

JavaScript provides various methods for manipulating strings:

  • String.length: Returns the length of a string.
  • String.toUpperCase(): Converts a string to uppercase.
  • String.toLowerCase(): Converts a string to lowercase.
  • String.concat(): Concatenates two or more strings.
  • String.indexOf(): Returns the index of the first occurrence of a specified value in a string.
  • String.substring(): Extracts characters from a string between two specified indices.
  • String.replace(): Replaces a specified value with another value in a string.
  • String.split(): Splits a string into an array of substrings.
