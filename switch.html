Switch Case in JavaScript

The switch statement is used to perform different actions based on different conditions. It evaluates an expression, matches the expression value with case labels, and executes the block of code associated with the matching case.

Syntax


        switch (expression) {
            case value1:
                // Statements
                break;
            case value2:
                // Statements
                break;
            ...
            default:
                // Statements
                break;
        }

The switch keyword is followed by an expression in parentheses. The expression is evaluated once and compared with the values of each case. If there is a match, the associated block of code is executed. If no match is found, the default block (if provided) is executed.

Example


        let day = "Wednesday";
        let message;

        switch (day) {
            case "Monday":
                message = "It's Monday!";
                break;
            case "Tuesday":
                message = "It's Tuesday!";
                break;
            case "Wednesday":
                message = "It's Wednesday!";
                break;
            default:
                message = "It's neither Monday nor Tuesday nor Wednesday.";
                break;
        }

        console.log(message); // Output: It's Wednesday!

In this example, the value of the day variable is evaluated in the switch statement. Since the value is "Wednesday", the corresponding case block is executed, assigning the message "It's Wednesday!" to the message variable.

Switch case statements provide a cleaner way to write code when there are multiple conditions to be evaluated.
